# BerozKala Backend Project
This program is webapi-based and written in C# (.NET). program is the backend of the [Berozkala Frontend](https://github.com/Alixlg/berozkala-frontend-project) project.

## Program

This project was generated using [.NET SDK](https://dotnet.microsoft.com/en-us/download/dotnet/10.0) version 10.0

## Prerequisites

- Install Dotnet Runtime or SDK version 10 or upper
- Install EF-Core
```bash
dotnet tool install --global dotnet-ef
```
```bash
dotnet restore
```
## How To Run

- 1> Create folder with `DbFiles` name in root directory

- 2> Enter in Terminal :
```bash
dotnet ef database update
```
```bash
dotnet build
```
- 3> Run again and enjoy :)
